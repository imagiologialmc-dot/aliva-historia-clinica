# Aliva Saúde — História Clínica

Formulário de história clínica para Imagiologia — Aliva Saúde.

## 🚀 Como instalar no Android

1. Acede ao link da app pelo **Chrome no Android**
2. Aparece um banner "Adicionar ao ecrã inicial" — toca em **Instalar**
3. Ou vai ao menu ⋮ → **Adicionar ao ecrã inicial**
4. A app fica instalada com o ícone da Aliva

## 🌐 Deploy via GitHub Pages

1. Faz upload de todos os ficheiros para um repositório GitHub
2. Vai a **Settings → Pages**
3. Em **Source**, seleciona `main` branch e pasta `/root`
4. O link da app será: `https://<teu-user>.github.io/<repo>/`

## 📁 Ficheiros

| Ficheiro | Função |
|---|---|
| `index.html` | Formulário principal |
| `manifest.json` | Configuração PWA (nome, ícone, cores) |
| `sw.js` | Service Worker (funciona offline) |
| `icon-192.jpg` | Ícone da app 192x192 |
| `icon-512.jpg` | Ícone da app 512x512 |
